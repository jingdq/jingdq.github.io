Android消息处理相关问题

  1. 在handler的post方法中设置一个执行较长任务的Runable对象（称为任务A），然后再这段代码 之后 同样执行一段任务代码（称为任务B），请问那个任务先执行？并解释原因。 例如：

    protected void onCreate(Bundle savedInstanceState) {
    super.onCreate(savedInstanceState);
    setContentView(R.layout.activity_main);
    mHandler.post(new Runnable() {
        @Override
        public void run() {

            try {
                for (int i = 0; i < 100; i++) {
                    Thread.sleep(1000);
                    Log.i("xxx", Thread.currentThread() + " i : " + i);

                }

            } catch (InterruptedException e) {
                e.printStackTrace();
            }

        }
    });
    try {
        for (int i = 0; i < 100; i++) {
            Thread.sleep(1000);
            Log.i("xxx", Thread.currentThread() + " j : " + i);

        }

    } catch (InterruptedException e) {
        e.printStackTrace();
    }

  }
